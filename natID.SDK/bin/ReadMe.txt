Extract the corresponding 'bin_*platform_date.7z' archive from
https://github.com/idzafic/natID/releases into this location.

For Windows:
 - place *.dll files directly into the bin folder
 - place *.lib files into the bin/lib folder

For macOS:
 - place *.dylib files into the bin/lib folder

For Linux:
 - place *.so files into the bin/lib folder

You can use one of the 'natIDSDKAllInOne_*platform_date.7z' archives
(from https://github.com/idzafic/natID/releases) as a reference to see
where each file should be placed.
